# Expenses Tracker - Beginner

A simple yet useful tool to help you track all of your daily expenses. It is made to showcase
the beginning of my journey with data retention after a process has ended. 

## How it works:

1. Clone or download the files into a designated folder on your local device.
2. Ensure that both files (main.py and data.json) are in the same folder at the same hierarchical level.
3. You can run the code in your terminal or your IDE of choice (PyCharm, VSCode, etc.).
4. Once started, just follow the onscreen instructions!

- Nested dictionary structure saved in JSON file.
- While loops used for continuous changes.
- Clearing functionality for repeated use.
- try/except used for correct input checking.
- Inputs are checked for typing errors.
